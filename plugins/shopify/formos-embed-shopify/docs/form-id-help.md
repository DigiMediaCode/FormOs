# Finding A FormOS Form ID

1. Log in to FormOS.
2. Open **Dashboard → Forms**.
3. Open the form you want to embed.
4. Use the Form ID shown in the embed section, or copy it from the embed URL:

```text
https://formos.com.au/embed/forms/FORM_ID
```

Paste that value into the Shopify **FormOS Form** app block.
