## Public Form Logo

Add a small centered logo above the public form title.

Use existing logo file if available:

public/formos-logo.png

If the logo file does not exist, skip it gracefully.

Logo requirements:

- centered
- small size
- placed above the form title
- should not dominate the page
- should have proper spacing below
- should look good on mobile

Suggested display size:

- max width: 90px to 130px
- height auto
